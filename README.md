## 목적

셸 스크립트는 따로 공부해야 할 만큼 중요는 한데, 시간은 없고, 그런 사람들을 위한 저장소를 만들어 보자는 생각에 시작

## What is ShellScript ?

Unix/Linux 또는 POSIX _(Portable Operating System Interface)_ 를 지원하는 운영체제인 MacOS 등에서 일반적으로 사용하는 명령어들과 프로그래밍적인 요소로 이루어진 인터프리터 기반의 스크립트 언어
